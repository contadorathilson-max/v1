# Sorteador de Times

App Android criado para sortear times de futebol de forma equilibrada.

## Funcionalidades

- Cadastro de jogadores.
- Escolha do nível do jogador de 1 a 5 estrelas por botões, sem depender do RatingBar.
- Edição do nível do jogador depois de cadastrado.
- Edição do nome do jogador.
- Remoção de jogador com confirmação.
- Escolha da quantidade de times.
- Escolha da quantidade de jogadores por time.
- Sorteio equilibrado pela soma das estrelas.
- Lista de reservas quando houver mais jogadores do que vagas.
- Registro de gols e assistências por jogador.
- Remoção de gols e assistências lançados por engano.
- Ranking de gols e assistências.
- Botão para copiar o último sorteio.
- Botão para zerar estatísticas.
- Botão para limpar a lista de jogadores.
- Salvamento local dos jogadores e estatísticas.
- Rodapé com o nome do desenvolvedor: Athilson Alves.

## Como gerar o APK

Este projeto já possui um workflow do GitHub Actions.

1. Suba os arquivos no GitHub.
2. Acesse a aba **Actions**.
3. Rode o workflow **Gerar APK Android**.
4. Baixe o artifact **SorteadorDeTimes-APK**.
5. Extraia o ZIP baixado e instale o APK no Android.
